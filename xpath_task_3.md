#  Sign Up Page XPath Selectors

---

## **Full Name Field**
```xpath
//input[@placeholder="Full Name"]
//input[@name="fullName"]
//input[@type="text" and contains(@placeholder, "Name")]
```

---

## **Email Address Field**
```xpath
//input[@type="email"]
//input[@placeholder="Email address"]
//input[@name="email"]
```

---

## **Password Field**
```xpath
//input[@type="password"]
//input[@placeholder="Password"]
//input[@name="password"]
```

---

## **Sign Up Button**
```xpath
//button[text()="Sign Up"]
//button[@type="submit"]
```
